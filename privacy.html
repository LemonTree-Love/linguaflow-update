LinguaFlow Privacy Policy

Last updated: April 23, 2026

Data Collection

LinguaFlow does not collect, transmit, or store any personal data on external servers. All user data — including vocabulary, settings, and API keys — is stored locally on the user's device using the Chrome storage.local API.

Cloud Sync

If the user explicitly enables cloud sync (WebDAV or iCloud), data is transmitted directly between the user's device and their own chosen storage service. LinguaFlow does not operate or have access to any intermediate server in this process.

Third-Party API Calls

When the user configures a translation provider (e.g. DeepSeek, OpenAI, DeepL, Google, Azure), the selected text and surrounding context are sent to the chosen third-party API for translation. These requests go directly from the user's browser to the third-party service. LinguaFlow does not proxy, log, or retain any of this data.

Permissions

The extension requests the following permissions solely for its stated functionality:

  • storage — save settings and vocabulary locally
  • activeTab — access page content to highlight words and show tooltips
  • contextMenus — add a right-click translation option
  • alarms — schedule sync and export tasks
  • notifications — notify when sync or export completes
  • host_permissions (all URLs) — run content scripts for word highlighting and translation on any webpage

Analytics

LinguaFlow does not include any analytics, tracking, or telemetry.
